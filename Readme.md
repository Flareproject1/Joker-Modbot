Joker Modbot
Бот-модератор для Telegram супергрупп.

Функции:
Капча для новых участников
Автоматическая проверка пользователей
Панель администратора с управлением пользователями
Бан, мут, разбан, информация о пользователе
Статистика группы (кол-во пользователей и проверенных)
Стек:
Python
Aiogram
SQLite (для хранения информации о пользователях)
Логирование через logging
